Base de dados para o funcionamento da LooKat

Este projeto é o Banco de Dados feito no Supabase para armazenar e enviar todos os conteúdos e informações dos usuários da plataforma LooKat. Esse projeto complemnta a plataforma, mas também é minha atividade final da matéria de Banco de Dados III

O Supabase é um banco de dados relacional que utiliza do PostegreSQL para criar seus bancos de dados. Além disso possui fácil integração com o FlutterFlow e garante serviços de Autenticação

As tabelas criadas são estruturadas para armazenar os dados dos usuários, dos conteúdos, das aulas, e assim por diante. Além disso, os Buckets garantem que os arquivos de imagem fiquem salvos de maneira segura no Banco de Dados
